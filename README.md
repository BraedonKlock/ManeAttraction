# Mane Attraction Hair Design

Welcome to the official website project for **Mane Attraction Hair Design**, a professional hair styling business based in Smiths Falls, Ontario.

A custom single-page application (SPA) built with vanilla HTML, CSS, and JavaScript. Features a modern, elegant design with smooth animations and seamless page transitions.

---

## Live Site

- Primary Domain: [https://maneattractionhairdesign.com](https://maneattractionhairdesign.com)

---

## Project Structure

```
/
├── index.html              # Main HTML file with header, footer, and navigation
├── JavaScript/
│   ├── main.js             # Routing, page transitions, and navigation
│   ├── helper.js           # Scroll animations, lightbox, and utilities
│   ├── home.js             # Home page content
│   ├── about.js            # About page content
│   ├── services.js         # Services page content
│   ├── bridal.js           # Bridal page content
│   ├── gallery.js          # Gallery page content
│   └── contact.js          # Contact page content
├── CSS/
│   └── modern.css          # All styles (variables, components, responsive)
├── Photos/                 # All image assets (logos, gallery, bridal, studio)
└── README.md               # Project documentation
```

---

## Features

- **Modern UI Design** — Clean, elegant interface with glassmorphism effects
- **Smooth Animations** — Scroll-triggered reveal animations and page transitions
- **Fully Responsive** — Optimized for mobile, tablet, and desktop
- **Bridal Section** — Dedicated bridal services showcase with image slider
- **Interactive Gallery** — Grid layout with lightbox functionality
- **Google Maps Integration** — Embedded map on contact page
- **Local Storage** — Remembers last visited page (with 5-minute expiration)
- **Preloader** — Elegant loading animation while assets load

---

## Technologies Used

- **HTML5**
- **CSS3** (Custom Properties, Flexbox, Grid, Animations)
- **JavaScript** (Vanilla ES6 Modules)
- **Google Fonts** (Cormorant Garamond, Montserrat)
- **Intersection Observer API** (Scroll animations)
- **Mobile-first Responsive Design**
